# FunkmapFront

Для установки необходимы Node.js, npm, angular-cli.

1.  Клонировать репозиторий
2.  Выполнить npm install
3.  Выполнить ng serve
4.  Для свзяи с <a href="https://github.com/rogulenkoko/funkmap-back">бекендом</a> заменить в файле environment.ts поле production на tru или на предыдущем шаге выполнить ng serve --prod
